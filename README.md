# Algorithm Problem Solving Repository

백준, Programmers, Project Euler, LeetCode **문제 풀이**를 정리하는 저장소입니다.
**Java로 알고리즘과 자료구조 학습 기록**을 남깁니다.

<br>


1️⃣ Baekjoon / Programmers

- 백준과 프로그래머스 문제는 `BaekjoonHub` 확장 프로그램을 통해 
[java_Coding_Challenges 리포지토리](https://github.com/yunzivv/java_Coding_Challenges)에 자동 커밋됩니다.

2️⃣ LeetCode / Project Euler

- BaekjoonHub에서 지원하지 않는 플랫폼(LeetCode, Project Euler)은 
[현재 리포지토리](https://github.com/yunzivv/Euler)에 직접 관리합니다.

## 📂 Project Structure

```
src/main/java/org/example

├── Main.java
│   └─ 백준(BOJ) 문제 풀이 실행용 클래스

├── Euler.java
│   └─ Project Euler 문제 풀이

├── leetCode.java
│   └─ LeetCode 문제 풀이

├── Programmers.java
│   └─ 프로그래머스 문제 풀이

└── Util.java
    └─ 알고리즘 문제 풀이에 사용하는 공통 유틸 함수
```
